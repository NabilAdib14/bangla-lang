##BanglaLang Interpreter

A lightweight, custom-built interpreter for a toy programming language designed from scratch. Supports integers, floats, variables, arithmetic, and control statements, with a modular architecture for easy extension. Perfect for learning compiler design, parsing, and AST evaluation.

Features:

Lexer and Parser for BanglaLang syntax

Abstract Syntax Tree (AST) implementation

Support for expressions, variables, and basic statements

Modular design for adding new language features

Example scripts included to test functionality

Tech Stack:
C++ | Object-Oriented Design | LL(1) Parsing | AST | Git
