# BanglaLang Interpreter

A lightweight, custom-built interpreter for a toy programming language designed from scratch.  
Supports **integers, floats, variables, arithmetic operations, and control statements**, with a modular architecture for easy extension.  

---

## Features
- Lexical analysis and parsing for BanglaLang syntax
- Abstract Syntax Tree (AST) evaluation
- Support for expressions, variables, and basic statements
- Modular design for adding new language features
- Example scripts included for testing

---

## Tech Stack
C++ | Object-Oriented Design | LL(1) Parsing | AST | Git

---

## Getting Started
1. Clone the repository
2. Compile the interpreter using your preferred C++ compiler
3. Run example `.bng` scripts or create your own
